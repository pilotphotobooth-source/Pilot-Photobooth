Installation en 5 étapes

1. Créez votre compte dans l'application mobile
2. Choisissez votre abonnement
3. Installez le logiciel sur le PC de la borne
4. Connectez-vous avec les mêmes identifiants
5. Votre borne apparaît en ligne dans 30 secondes ✅

Questions fréquentes
Compatible macOS ?+
Non, Windows 10/11 uniquement pour le logiciel. L'app mobile est Android et iOS.
Quel logiciel photo compatible ?+
dslrBooth, Sparkbooth, Breeze — tout logiciel qui enregistre en JPG/PNG dans un dossier Windows.
Plusieurs bornes possibles ?+
Oui ! Chaque borne supplémentaire = 6€/mois. Toutes visibles dans la même application.
